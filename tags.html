Important Html tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we will learn how to make HTML Tables via table and related tags.

The tables below do not include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 Headings Includes a heading (level 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong Bold make bold text <strong>technical sophistication</strong> technical sophistication
a anchor link make a link to another asset <a>technical sophistication</a> technical sophistication
img image embed an image <img>technical sophistication</img>
code code snippet Adjusts font to indicate a sode snippet <code>technical sophistication</code> technical sophistication
